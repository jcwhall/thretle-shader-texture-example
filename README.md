# Thretle Textured Shader example

I couldn't find a basic example of [Sveltekit](https://svelte.dev/kit/) + [Threlte](https://threlte.xyz/) + Loading a Texture + Shader.
This is it.

- Texture is in the static folder (from Threejs example)
- Simple shaders

## Example ##
See an example [here](https://jcwhall.github.io/thretle-shader-texture-example/)

## Install ##
Run: 
```sh
npm install
```

## Running ##

To run:
```sh
npm run dev
```

Or run and open the app in a new browser tab:
```sh
# 
npm run dev -- --open
```
